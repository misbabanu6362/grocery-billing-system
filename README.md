# Grocery-billing-system
Grocery Billing System

A beginner Python project that simulates a simple grocery store billing system.

Features
Product catalog using Dictionary
Add products to bill
Enter quantity of products
Calculate total amount
Validate product availability
Apply 10% discount on bills above ₹500
Generate final bill

Concepts Used
Variables
Input and Output
Conditional Statements
Loops
Dictionaries
Lists
Arithmetic Operators
